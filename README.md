# 🎓 GEC Student-Faculty Management System

A comprehensive Flutter-based academic management platform designed to streamline communication, performance tracking, and administrative operations within educational institutions.

The application provides dedicated dashboards for Students, Faculty Members, and Administrators, enabling efficient academic monitoring, grade analysis, and student performance management through an intuitive and modern user interface.

---

# 📌 Overview

The GEC Student-Faculty Management System is designed to simplify academic workflows by providing centralized access to student records, semester results, performance analytics, and administrative controls.

The platform supports multiple user roles and offers data-driven insights through interactive charts, filters, and dashboards.

---

# ✨ Key Features

## 👨‍🎓 Student Dashboard

* View semester-wise results
* Track CGPA and academic progress
* Access subject-wise grades
* Monitor backlog status
* View performance trends across semesters
* Clean and user-friendly interface

## 👨‍🏫 Faculty Dashboard

* View student academic records
* Analyze class performance
* Access student grade reports
* Monitor subject-wise statistics
* Track academic progress of students

## 👨‍💼 Admin Dashboard

* Full access to academic data
* Manage student records
* Manage faculty information
* Monitor institution-wide performance
* Generate academic insights and reports
* Access analytics and visualizations

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/aksharadileep/GEC-Student-Faculty-app.git
```

### Navigate to Project

```bash
cd GEC-Student-Faculty-app
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```
